
# 暗影焰刀重设计产品方案

## 方案名：影焰回刃

## 1. 设计目标

将暗影焰刀从原本的“投掷飞刀”改造成一把具有明确操作循环的武器：

> **左键高速穿刺布刀，右键召回飞刀收割，多刀穿过同一敌人时触发影焰爆炸。**

这套设计的重点不是单纯提高伤害，而是让玩家通过**命中、布置、走位、召回时机**来打出更高收益。

核心关键词：

```text
高速
直线
穿刺
驻留
召回
爆发
紫黑影焰
```

---

# 2. 核心玩法概述

## 左键：影焰穿刃

玩家左键攻击时，向鼠标方向投出一把高速暗影焰飞刀。

飞刀具有以下特征：

```text
飞行速度更快
不受重力影响
沿直线飞行
命中敌人后穿过敌人
穿过过程造成伤害
穿过后在敌人身后留下驻留飞刀
```

飞刀不再是慢速下坠的投掷物，而是一把更接近“高速刺穿敌人”的影焰飞刃。

---

## 右键：万刃归影

玩家右键时，场上所有属于自己的驻留飞刀会被同时召回。

召回过程中：

```text
所有驻留飞刀向玩家所在方向飞回
召回路径上的敌人会受到伤害
召回后飞刀消失
```

如果同一次召回中，同一个敌人被 **4 把或更多**召回飞刀命中，则在该敌人位置触发一次影焰爆炸。

这里的“超过三把刀”统一定义为：

```text
4 把及以上
```

---

# 3. 完整战斗循环

玩家体验应该是：

```text
左键投出高速飞刀
↓
飞刀直线穿过敌人并造成伤害
↓
飞刀停在敌人身后，形成驻留飞刀
↓
玩家继续攻击，积累多把驻留飞刀
↓
玩家移动到合适位置
↓
右键召回所有驻留飞刀
↓
飞刀回收路径再次切割敌人
↓
4 把或更多飞刀命中同一敌人时触发影焰爆炸
```

这套循环的核心是：

> **驻留飞刀的位置由左键命中方向决定，召回路径由玩家站位决定。**

玩家越会调整站位，越容易让多把飞刀同时穿过同一个敌人，从而触发爆炸。

---

# 4. 左键攻击规则

## 4.1 飞刀基础规则

左键发射的飞刀应满足以下产品要求：

| 项目       | 规则                           |
| ---------- | ------------------------------ |
| 飞行轨迹   | 直线飞行                       |
| 重力       | 不受重力影响                   |
| 速度       | 明显快于原版暗影焰刀           |
| 主要作用   | 穿刺敌人并生成驻留飞刀         |
| 命中敌人   | 造成穿刺伤害                   |
| 命中后     | 飞刀穿过敌人，并在敌人身后驻留 |
| 命中地形   | 消失，不生成驻留飞刀           |
| 未命中敌人 | 飞行一段距离后消失             |

---

## 4.2 穿刺伤害

飞刀命中敌人时，不应只是“生成驻留刀”的触发器。

它本身在穿过敌人时必须造成伤害。

产品表现应是：

```text
飞刀刺入敌人
↓
穿过敌人身体
↓
敌人受到一次穿刺伤害
↓
飞刀出现在敌人身后并停驻
```

这能让玩家明确感受到：

> 这把刀真的穿过了敌人，而不是碰到敌人后凭空生成了一把刀。

---

## 4.3 驻留飞刀生成位置

驻留飞刀生成在敌人身后。

“身后”不按敌人的朝向判断，而是按飞刀的飞行方向判断。

示例：

```text
飞刀从左往右命中敌人
→ 驻留飞刀出现在敌人右侧

飞刀从右往左命中敌人
→ 驻留飞刀出现在敌人左侧

飞刀从上往下命中敌人
→ 驻留飞刀出现在敌人下方
```

这样玩家能直观理解：

> 飞刀从哪里刺进去，就会从另一侧穿出来并停住。

---

## 4.4 每把左键飞刀的生成限制

首版建议：

```text
每把左键飞刀最多生成 1 把驻留飞刀。
```

也就是说，飞刀命中第一个有效敌人后，就完成穿刺并在目标身后驻留，不继续穿过多个敌人生成多把刀。

原因：

```text
避免驻留飞刀增长过快
避免右键爆发失控
降低画面混乱
方便玩家理解场上飞刀来源
```

后续如果觉得清怪能力不足，可以再考虑增强，但首版不建议开放多目标生成。

---

# 5. 驻留飞刀规则

## 5.1 基础规则

飞刀穿过敌人后，会在敌人身后形成一把驻留飞刀。

驻留飞刀规则如下：

| 项目             | 规则                           |
| ---------------- | ------------------------------ |
| 是否移动         | 不移动，停在原地               |
| 是否跟随敌人     | 不跟随                         |
| 是否主动攻击     | 不主动攻击                     |
| 是否造成持续伤害 | 不造成                         |
| 是否阻挡单位     | 不阻挡玩家或敌人               |
| 是否可召回       | 可以被右键召回                 |
| 消失方式         | 超时、被召回、超出上限后被替换 |

驻留飞刀的作用是作为后续右键召回的资源，而不是一个持续伤害单位。

---

## 5.2 驻留飞刀上限

必须设置飞刀上限。

首版推荐：

```text
每名玩家最多同时拥有 12 把驻留飞刀。
```

这个数值的理由：

```text
4 把命中即可触发爆炸
6 把上限提供一定容错
不会让玩家无限预存爆发
不会让画面被飞刀铺满
方便控制 Boss 战强度
```

---

## 5.3 超过上限时的处理

当玩家已经拥有 12 把驻留飞刀时，如果新的驻留飞刀生成，则采用：

```text
新飞刀生成
最早生成的旧飞刀消失
```

也就是“新刀替换最旧刀”。

不建议采用“达到上限后不再生成”的方式，因为玩家会感觉命中后没有反馈。

替换旧刀时，旧刀只消失，不造成伤害，不触发爆炸。

---

## 5.4 驻留时间

首版推荐：

```text
驻留飞刀存在约 6 秒。
```

6 秒的目的：

```text
给玩家足够时间继续布刀
给玩家足够时间调整站位
避免玩家长期囤积爆发
让战斗节奏保持紧凑
```

驻留飞刀即将消失时，需要有明显但不刺眼的反馈，例如：

```text
透明度降低
轻微闪烁
暗焰逐渐变弱
```

玩家应该能看出哪些刀快要消失。

---

# 6. 右键召回规则

## 6.1 召回触发

玩家右键时，召回当前场上所有属于自己的驻留飞刀。

规则如下：

| 项目               | 规则                         |
| ------------------ | ---------------------------- |
| 召回对象           | 当前所有驻留飞刀             |
| 最少数量           | 有 1 把也可以召回            |
| 没有驻留飞刀时     | 不造成伤害，只给很轻的空反馈 |
| 召回后             | 所有被召回飞刀消失           |
| 是否保留未召回飞刀 | 不保留，右键会召回全部       |

右键不应是单纯的引爆按钮，而是“把场上的刀全部拉回”。

---

## 6.2 召回路径

召回飞刀从原本驻留位置飞回玩家附近。

产品预期是：

```text
玩家按下右键时的站位，决定主要召回路径。
```

也就是说，玩家需要在右键前主动调整位置。

如果玩家站在敌人的另一侧，飞刀回收时就更容易穿过敌人；如果玩家站位不好，飞刀可能直接飞回而不命中。

这能形成明确的操作深度：

```text
先布刀
再走位
最后召回
```

---

## 6.3 召回伤害

召回过程中，每把飞刀在路径上可以对敌人造成伤害。

规则如下：

```text
召回飞刀可以穿过敌人
召回飞刀命中敌人时造成召回伤害
每把召回飞刀对同一个敌人最多造成一次召回伤害
召回伤害低于左键穿刺伤害
```

这样右键召回的收益来自多把飞刀的组合，而不是单把飞刀过强。

---

## 6.4 召回是否穿过地形

建议产品规则为：

```text
左键飞刀会被地形阻挡。
右键召回飞刀可以穿过地形。
```

原因：

```text
左键被地形阻挡，符合投掷武器直觉。
右键召回如果被地形阻挡，容易导致飞刀丢失或手感不稳定。
```

右键召回的重点是“收回已经布置好的刀”，不应该频繁被墙体破坏体验。

---

# 7. 四刀爆炸规则

## 7.1 触发条件

爆炸条件定义为：

```text
同一次右键召回中，
同一个敌人被 4 把或更多不同的召回飞刀命中，
触发一次影焰爆炸。
```

注意：

```text
只统计正在召回的驻留飞刀。
左键飞刀命中不计入爆炸条件。
爆炸伤害不计入爆炸条件。
同一把召回飞刀不能重复计数。
```

---

## 7.2 “同时命中”的产品定义

不要求 4 把飞刀在完全同一瞬间命中。

产品上建议定义为：

```text
同一次召回中，
同一个敌人在一个很短的时间窗口内被 4 把不同飞刀命中，
就视为“多刀同时命中”。
```

首版建议体感窗口：

```text
约 0.15～0.25 秒。
```

这个窗口的目的是给玩家一点容错，避免视觉上明显是多刀穿过敌人，但系统没有触发爆炸。

---

## 7.3 爆炸次数限制

为了控制强度，必须限制爆炸次数：

```text
同一个敌人每次召回最多触发 1 次爆炸。
```

示例：

```text
玩家召回 6 把飞刀。
同一个 Boss 被 4 把飞刀命中，触发爆炸。
后续第 5、6 把飞刀仍然造成召回伤害。
但不会再次触发爆炸。
```

---

## 7.4 多敌人情况

如果一次召回中，多名敌人都满足 4 刀命中条件，则可以分别触发爆炸。

规则：

```text
每个敌人每次召回最多爆炸 1 次。
不同敌人之间互不影响。
```

这样对群战斗会更爽，但单个 Boss 不会因为反复爆炸而失控。

---

## 7.5 爆炸伤害定位

爆炸是这套武器的主要奖励，但不能成为无条件高伤害。

伤害结构建议：

```text
左键穿刺伤害：稳定输出
召回单刀伤害：补充伤害
四刀爆炸伤害：核心爆发奖励
```

爆炸应该奖励：

```text
命中率
布刀数量
召回角度
玩家站位
右键时机
```

而不是奖励玩家无脑按右键。

---

# 8. 推荐首版数值方向

以下是产品调试用的首版建议，不是最终平衡值。

| 项目                       | 首版建议                       |
| -------------------------- | ------------------------------ |
| 驻留飞刀上限               | 12 把                          |
| 驻留时间                   | 约 6 秒                        |
| 左键飞刀轨迹               | 直线                           |
| 左键飞刀重力               | 无重力                         |
| 左键飞刀速度               | 明显快于原版                   |
| 每把左键飞刀生成驻留刀数量 | 最多 1 把                      |
| 召回触发                   | 右键召回全部驻留飞刀           |
| 召回单刀伤害               | 左键穿刺伤害的 50%～65%        |
| 爆炸触发门槛               | 4 把或更多召回飞刀命中同一敌人 |
| 爆炸伤害                   | 左键穿刺伤害的 180%～240%      |
| 爆炸范围                   | 中小范围，不做清屏             |
| 单敌人爆炸限制             | 每次召回最多 1 次              |
| 额外暗影焰奖励             | 不加入                         |

---

# 9. 不加入的机制

本版明确不加入以下内容：

```text
不加入暗影焰状态额外奖励
不让驻留飞刀跟随敌人
不允许无限囤积飞刀
不让单个敌人一次召回内重复爆炸
不让左键飞刀受重力影响
不保留原版下坠手感
不把右键做成无条件引爆按钮
```

关于暗影焰效果，需要注意：
如果武器原本有基础暗影焰主题或常规暗影焰效果，可以按平衡需求保留。

# 10. 视觉表现需求

## 10.1 左键飞刀

左键飞刀使用原有贴图，只是加快直线飞行速度。

## 10.2 穿刺反馈

飞刀穿过敌人时，需要清楚反馈。

建议表现：

敌人身上出现短暂紫色斩痕
飞刀从敌人另一侧穿出
驻留刀稳定悬停

玩家必须一眼看懂：

## 10.3 驻留飞刀

驻留飞刀是玩家判断资源数量的重要视觉元素。

建议表现：

半透明紫黑刀影
边缘有少量暗焰粒子
亮度中等，不抢屏幕
即将消失时逐渐变淡

## 10.4 右键召回

召回是整套武器的爽点，表现必须明显。

建议表现：

```text
所有驻留飞刀同时被拉回
飞刀拖出细长紫黑斩线
回收速度快
命中敌人时有短促闪光
多刀穿过敌人时反馈更强
```

玩家按下右键时，应该感受到：

> 场上的刀被我一瞬间收回来了。

---

# 11. 音效反馈需求

音效可以分为四层。

## 左键投刀

```text
短促、锋利的破空声
速度感强
声音不厚重
```

## 穿刺命中

```text
轻微刺入声
带一点暗焰灼烧感
命中反馈清楚但不吵
```

## 右键召回

```text
多把刀同时回收的撕裂声
比左键更低沉
有明显收束感
```

## 四刀爆炸

```text
短促的影焰爆裂声
带刀刃撕裂或碎裂感
持续时间短
```

---

# 12. 边界情况规则

## 12.1 没有驻留飞刀时右键

没有动作

## 12.2 敌人被左键飞刀击杀

如果左键飞刀命中并击杀敌人，仍然可以在该敌人身后生成驻留飞刀。

原因：

```text
保证命中反馈一致
避免玩家因为击杀敌人反而少一把刀
让布刀节奏更稳定
```

---

## 12.3 驻留飞刀到期

驻留飞刀到期后自然消失。

规则：

```text
不造成伤害
不触发爆炸
不进行召回
```

---

## 12.4 切换武器或玩家死亡

建议规则：

```text
玩家死亡时，驻留飞刀全部消失。
玩家切换到其他武器后，驻留飞刀快速消散，不能继续召回。
```

原因：

```text
避免提前布刀后切换武器打爆发
避免跨武器利用机制
保持暗影焰刀自身完整性
```

---

## 12.5 多人游戏

多人情况下：

```text
每名玩家只管理自己的驻留飞刀。
每名玩家只能召回自己的驻留飞刀。
爆炸判定只统计该玩家本次召回的飞刀。
不同玩家的飞刀数量和爆炸判定互不影响。
```

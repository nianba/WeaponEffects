# 长矛类武器枪势连招项目文档

## 1. 当前项目定义

为长矛 / 枪 / 戟类近战武器增加专属枪势动作，替代原版重复单一刺击。

当前仓库中已经跑通的普通左键是三段循环：

```text
上挑 -> 绕身回扫 -> 终结刺击
```

旧版“扎”动作仍在 `SpearActionScheme` 中保留为 `ForwardThrust`，但当前普通左键循环从 `SpearActionScheme.FirstEnabledStepIndex = 1` 开始，因此它不属于当前启用动作。

长矛表现重点仍然是：

```text
枪尖
枪身
蓄势
贯穿
长柄挥扫
```

避免做成刀剑类连续月牙斩或大范围剑气。

## 2. 当前动作数据结构

旧的 `TridentSpearComboScheme` 已删除。当前动作定义集中在：

```text
Core/Spears/SpearActionScheme.cs
```

每段动作为一个 `SpearActionStep`，由五类数据组成：

```text
Timing    = 有效帧、起手结束、攻击结束、挥舞音效触发进度
Gameplay  = reach、碰撞宽度、伤害倍率、时间倍率、extraUpdates
Motion    = 弧线或突刺轨迹参数
Afterimage = 挥扫弧面视觉与碰撞采样
TipGlow   = 枪尖锋芒视觉与碰撞延伸
```

这种结构是当前后续调参的准入口。动作设计变更应优先改 `SpearActionScheme` 中对应 step 的数据，而不是在 projectile 里写新的分支。

## 3. 基础方向规则

以下描述默认角色面向右侧。

```text
身前 = 鼠标方向 / 攻击方向
身后 = 鼠标反方向
身前上方 = 角色前方上侧
身后低位 = 角色背后偏下蓄枪位置
```

角色面向左侧时，整套动作水平镜像。

当前三段的关系：

```text
第一段：枪尖从身前挑起，制造纵向起势。
第二段：枪尖绕身体外侧回扫，形成长柄挥扫感。
第三段：从蓄势姿态向身前终结刺击，集中枪尖锋芒。
```

## 4. 输入与连招推进

当前普通左键连招按三段循环：

```text
第 1 段：上挑
第 2 段：绕身回扫
第 3 段：终结刺击
```

代码事实：

```text
SpearActionScheme 定义 4 个 step。
SpearActionScheme.FirstEnabledStepIndex = 1。
WeaponEffectsPlayer 从 step 1 开始消费。
实际循环为 1 -> 2 -> 3 -> 1。
```

停止攻击超过连招保留时间后，回到第一段上挑。

当前代码中虽然存在地面 / 空中终结分支枚举，但 `SelectFinisherBranch` 固定返回 `GroundedFinisher`，所以当前不会触发空中下砸。

## 5. 数值继承规则

长矛动作以武器原本属性为基础进行倍率缩放：

```text
左键实际伤害 = 原武器最终伤害 × 全局伤害倍率 × 当前动作伤害倍率 × 近战攻速伤害折算
左键实际动作间隔 = 原武器 useAnimation × 全局间隔倍率 × 当前动作时间倍率
实际击退 = 原武器击退 × 全局击退倍率
```

长矛左键不把近战攻速加成转换为更高出手频率，而是转换为伤害和额外暴击：

```text
近战攻速伤害折算 = 1 + (meleeAttackSpeed - 1) × 0.6
额外暴击率 = max(0, round((meleeAttackSpeed - 1) × 20))
```

低于 `1` 的近战攻速只降低折算伤害，不产生负暴击。

当前使用的全局配置复用刀剑配置：

```text
NormalSlashDamageMultiplier
NormalSlashIntervalMultiplier
SlashKnockbackMultiplier
ComboResetDelay
```

这种复用是当前成功路线的一部分，暂时不拆出长矛专用配置。

需要尽量继承的原武器属性包括：

```text
基础伤害
近战伤害加成
近战攻速加成
暴击率
击退
武器前缀
饰品加成
药水加成
原有命中特效
原有减益效果
```

原版特殊弹幕不直接保留原版 `Shoot`，后续通过显式映射手动补发。

## 6. 当前三段动作数值表

| 实际段数 | Step | 动作 | 伤害倍率 | 时间倍率 | 功能定位 |
| --- | --- | --- | --- | --- | --- |
| 1 | `RisingLift` | 上挑 | 2.3x | 1.8x | 起势、扩大纵向打击范围 |
| 2 | `Backsweep` | 绕身回扫 | 2.3x | 1.8x | 长柄挥扫、回枪、清身前区域 |
| 3 | `Finisher` | 终结刺击 | 3.0x | 2.3x | 终结、贯穿、枪尖爆发 |

时间倍率越大，动作越慢。

当前 `Finisher` 还定义了 `AirborneTimeMultiplier = 1.2f`，但由于空中分支暂未启用，普通实战里不会用到这个倍率。

## 7. 保留但未启用动作：ForwardThrust

`ForwardThrust` 当前参数：

```text
Timing:
  ActiveStart = 0.2
  ActiveEnd = 0.55
  WindupEnd = 0
  AttackEnd = 1
  SwingSoundProgress = 0.2

Gameplay:
  ReachScale = 1.15
  CollisionWidth = 22
  DamageMultiplier = 0.85
  TimeMultiplier = 0.65
  ExtraUpdates = 4

Afterimage:
  Disabled(collisionWidth: 30)

TipGlow:
  Enabled = true
  StartProgress = 0
  CenterOffset = 5
  BaseSize = 10
  GrowthSize = 30
  WidthScale = 0.4
  LengthScale = 1
  UniformScale = 1
  CollisionPadding = 6
```

它适合作为短前刺或快速起手，但当前普通连招没有使用。

后续如果要恢复四段连招，应先讨论它是否真的改善节奏。当前三段路线已经更直接地把视觉重点放在挥扫和终结上，不建议只因为代码里保留了 step 0 就恢复它。

## 8. 动作一：上挑

### 动作定义

枪尖从身前中低位挑至身前上方，形成纵向起势。

```text
动作幅度较大
枪尖路径清晰
用于建立后续回扫的高位起点
```

### 当前参数

```text
Timing:
  ActiveStart = 0.42
  ActiveEnd = 0.76
  WindupEnd = 0.48
  AttackEnd = 0.76
  SwingSoundProgress = 0.5

Gameplay:
  ReachScale = 1.05
  CollisionWidth = 24
  DamageMultiplier = 2.3
  TimeMultiplier = 1.8
  ExtraUpdates = 4

Motion:
  Style = Arc
  StartAngle = 60
  ReleaseAngle = 30
  EndAngle = -210
  RecoveryAngle = -210
  RadiusScale = 0.86 -> 0.9 -> 0.9 -> 0.86
```

### 判定区域

```text
角色身前到身前上方的斜向 / 弧形区域
```

碰撞采样来自 step.Afterimage：

```text
CollisionSampleCount = 8
CollisionSampleSpacing = 0.055
CollisionWidth = 32
HitboxScale = 1.1
```

### 视觉表现

```text
使用 Mhd + SlashTex + 配色源贴图绘制弧面残影
残影较长、较淡
不绘制枪尖强锋芒
命中爆光复用刀剑命中效果
```

当前上挑挥扫残影参数：

```text
VisualSampleCount = 18
VisualProgressWindow = 0.34
VisualWidth = 22
VisualAlpha = 0.014
VisualInnerShaftAmount = 0.54
VisualReachScale = 1.6
```

## 9. 动作二：绕身回扫

### 动作定义

枪尖绕身体外侧回扫，形成长柄武器的横向扫击感。

这个动作不是刀剑的大月牙斩，而是借长柄挥扫制造更长的棍身 / 枪身运动轨迹。

### 当前参数

```text
Timing:
  ActiveStart = 0.38
  ActiveEnd = 0.78
  WindupEnd = 0.48
  AttackEnd = 0.78
  SwingSoundProgress = 0.46

Gameplay:
  ReachScale = 1
  CollisionWidth = 28
  DamageMultiplier = 2.3
  TimeMultiplier = 1.8
  ExtraUpdates = 4

Motion:
  Style = Arc
  StartAngle = 150
  ReleaseAngle = 180
  EndAngle = 510
  RecoveryAngle = 510
  RadiusScale = 0.9 -> 0.86 -> 0.9 -> 0.84
```

### 判定区域

```text
主要判定覆盖回扫路径
判定跟随枪尖历史采样
视觉上允许比实际武器贴图更长
```

当前碰撞采样：

```text
CollisionSampleCount = 8
CollisionSampleSpacing = 0.06
CollisionWidth = 36
HitboxScale = 1.1
```

### 视觉表现

```text
继续使用 Mhd + SlashTex + 配色源贴图
残影比上挑更长
透明度保持低，避免脏和色块堆积
不绘制枪尖强锋芒
```

当前回扫挥扫残影参数：

```text
VisualSampleCount = 30
VisualProgressWindow = 0.46
VisualWidth = 30
VisualAlpha = 0.014
VisualInnerShaftAmount = 0.16
VisualReachScale = 1.6
```

## 10. 动作三：终结刺击

### 动作定义

终结段从蓄势姿态向身前刺出，集中表现枪尖锋芒和贯穿感。

当前代码实际使用 `GroundedFinisherTip`：

```text
身后 / 侧后蓄枪
向身前远端刺出
结束时停在身前远点
```

### 当前参数

```text
Timing:
  ActiveStart = 0.37
  ActiveEnd = 0.86
  WindupEnd = 0.48
  AttackEnd = 0.7
  SwingSoundProgress = 0.45

Gameplay:
  ReachScale = 1.35
  CollisionWidth = 32
  DamageMultiplier = 3
  TimeMultiplier = 2.3
  ExtraUpdates = 4
  AirborneTimeMultiplier = 1.2

Motion:
  Style = Thrust
  StartAngle = 150
  ReleaseAngle = 90
  RadiusScale = 0.9 -> 0.82
  EndXScale = 1.25
  EndY = 8
```

### 判定区域

```text
主判定是枪柄到枪尖的线段
枪尖锋芒会额外延伸碰撞距离
```

终结段基础碰撞：

```text
Afterimage = Disabled(collisionWidth: 42)
HitboxScale = 1
```

枪尖光 profile 会增加额外碰撞延伸。

### 视觉表现

```text
重点是枪尖锋芒
不绘制挥扫弧面
命中时复用当前命中闪光和音效
```

当前终结段枪尖锋芒参数：

```text
StartProgress = 0.62
CenterOffset = 50
BaseSize = 10
GrowthSize = 10
WidthScale = 3.0
LengthScale = 1.45
UniformScale = 1.1
CollisionPadding = 50
```

## 11. 右键蓄力投掷

当前 `SpearGlobalItem` 支持右键：

```text
右键按住 -> SpearThrowChargeProjectile
松开且达到最短蓄力 -> SpearThrowProjectile
```

蓄力时间受近战攻速影响：

```text
EffectiveFullChargeFrames = BaseFullChargeFrames / meleeAttackSpeed
```

当前投掷伤害、距离、视觉宽度、碰撞宽度随蓄力进度线性增长：

```text
DamageMultiplier: 1x -> 10x
TravelDistance: 1.5 屏 -> 5 屏
VisualWidth: 18 -> 34
CollisionWidth: 34 -> 58
```

全长矛适配第一版可先保留这套通用投掷，不逐把差异化。

## 12. 特效统一规则

当前长矛特效分为：

```text
挥扫弧面
枪尖锋芒
枪杆 / 空中终结残影
命中爆点
右键蓄力与投掷光效
```

已验证成功路线：

```text
挥扫弧面 = Mhd + SlashTex + 干净配色源贴图
枪尖锋芒 = SharpTears 额外贴图 + profile 主色
命中爆点 = 复用 SlashHitEffectProjectile
```

不要回到以下方案：

```text
大面积 SpearTipTrail 贴图
武器贴图采样作为残影颜色
纯 SlashTex 双贴图采样
高 alpha 多层叠加
```

这些方案之前已经暴露过白块、纯白、脏、色块堆积等问题。

## 13. 武器特殊效果挂载规则

原版长矛特殊效果后续按显式映射接入，不做自动猜测。

通用原则：

```text
原本在枪尖触发的效果，挂载到当前动作枪尖位置。
原本命中触发的减益，优先依赖 ItemLoader.OnHitNPC 保留。
原本由 projectile AI 生成的特色，需要单独重写或手动补发。
原本发射的枪尖弹幕，不直接读取 HeldItem.shoot 通用补发。
```

特殊武器第一版可以先只接管动作和配色，特色弹幕后续逐把确认。

## 14. 当前最终动作定义

当前长矛左键动作定义为：

```text
第一段：上挑
枪尖从身前挑至身前上方，制造纵向起势，并使用淡弧面残影表现长柄运动。

第二段：绕身回扫
枪尖绕身体外侧回扫，形成更长、更淡的挥扫残影，突出长柄武器的扫击范围。

第三段：终结刺击
从蓄势姿态向身前刺出，重点表现枪尖锋芒、贯穿感和命中反馈。
```

后续如果要恢复“扎”或空中下砸，应作为新一轮动作设计处理，不应混在当前全长矛适配第一版里。

# 长矛类武器枪势连招项目文档

## 1. 项目定义

为长矛 / 枪 / 戟类近战武器增加专属四段连招动作，替代原本重复单一刺击的攻击表现。

核心连招为：

```text
扎 → 上挑 → 绕身回扫 → 地面进尺 / 空中抡枪下砸
```

整体动作逻辑：

```text
扎定枪线
上挑起势
回扫蓄枪
地面进尺破阵
空中抡枪下砸
```

长矛类武器的表现重点是：

```text
枪尖
枪身
蓄势
贯穿
抡砸
```

避免做成刀剑类的连续月牙斩或大范围剑气。

---

# 2. 基础方向规则

以下描述默认角色面向右侧。

```text
身前 = 鼠标方向 / 攻击方向
身后 = 鼠标反方向
身前正上 = 角色前方上侧
身后低位 = 角色背后偏下蓄枪位置
```

角色面向左侧时，整套动作水平镜像。

第三段和第四段的关键方向关系：

```text
第三段：
枪尖从身前正上绕身体外侧回到身后低位。

第四段地面：
保持身后低位蓄枪姿态，蓄力后向身前进尺。

第四段空中：
保持身后低位蓄枪姿态，整根枪以枪尾 / 持握端为原点，从身后绕过人物头顶，向身前下砸。
```

---

# 3. 输入与连招推进

长矛类武器连续左键攻击时，按顺序推进四段动作：

```text
第 1 段：扎
第 2 段：上挑
第 3 段：绕身回扫
第 4 段：根据角色状态分支
```

第四段开始瞬间判断角色状态：

```text
角色在地面：第四段为地面进尺
角色在空中：第四段为空中抡枪下砸
```

第四段分支一旦确定，本次动作期间不再切换。

```text
空中进入第四段后，即使动作过程中落地，也继续完成抡枪下砸。
地面进入第四段后，即使动作过程中离地，也继续完成进尺。
```

第四段结束后，连招回到第一段。

中断规则：

```text
停止攻击超过连招保留时间后，回到第一段。
切换武器后，回到第一段。
玩家被强制打断后，回到第一段。
```

---

# 4. 数值继承规则

四段动作不单独写死武器强度。

所有动作以武器原本属性为基础进行倍率缩放：

```text
实际伤害 = 原武器最终伤害 × 当前动作伤害倍率
实际动作时间 = 原武器攻击时间 × 当前动作时间倍率
实际击退 = 原武器击退 × 当前动作击退倍率
```

需要继承的原武器属性包括：

```text
基础伤害
近战伤害加成
近战攻速加成
暴击率
击退
武器前缀
饰品加成
药水加成
原有命中特效
原有减益效果
原有枪尖射弹或特殊弹幕
```

连招系统只改变动作结构、轨迹、判定和视觉表现，不改变武器本身的成长定位。

# 5. 四段动作数值表

| 段数 |         动作 | 伤害倍率 | 时间倍率 | 击退倍率 | 功能定位         |
| ---- | -----------: | -------: | -------: | -------: | ---------------- |
| 1    |           扎 |    0.85x |    0.75x |    0.90x | 快速起手         |
| 2    |         上挑 |    1.00x |    0.90x |    1.00x | 起势、打上方     |
| 3    |     绕身回扫 |    0.85x |    1.05x |    1.20x | 扫开、回枪、蓄势 |
| 4A   |     地面进尺 |    1.30x |    1.15x |    1.15x | 终结、贯穿       |
| 4B   | 空中抡枪下砸 |    1.30x |    1.20x |    1.25x | 空中终结、下砸   |

时间倍率越大，动作越慢。

动作时间需要设置上下限，避免极端攻速导致动作看不清或过于拖沓。

# 6. 动作一：扎

## 动作定义

快速向身前点刺。

```text
起手短
出枪快
收招快
攻击线窄
```

## 枪尖路径

```text
身前中位 → 身前远点 → 回收
```

## 判定区域

```text
角色身前窄直线区域
```

## 视觉表现

```text
短直线枪芒
枪尖点状闪光
小型命中火花
少量速度线
```

## 命中反馈

```text
敌人身上出现点状穿刺光
反馈短促
不产生大爆炸
```

---

# 7. 动作二：上挑

## 动作定义

枪尖从身前中位挑至身前正上方。

```text
承接第一段
制造纵向动作
为第三段回扫提供高位起点
```

## 枪尖路径

```text
身前中位 → 身前上方 → 身前正上
```

## 判定区域

```text
角色身前到身前上方的斜向区域
```

## 视觉表现

```text
斜向上枪芒
细弧形上挑轨迹
向上飞散的火花
枪尖轨迹亮
枪杆残影淡
```

## 命中反馈

```text
命中点向上爆开
敌人有被挑起的视觉感
不做强制大幅击飞
```

---

# 8. 动作三：绕身回扫

## 动作定义

从第二段结束的身前正上位置开始，枪尖绕身体外侧回到身后低位。

这是整套连招的关键过渡动作。

第三段不是单纯横扫，而是：

```text
回枪
扫开
蓄势
```

## 枪尖路径

```text
身前正上
→ 绕身体外侧
→ 经过身体侧面
→ 回到身后低位
```

## 结束姿态

第三段结束时必须满足：

```text
枪尖在身后
枪尖微微下压
枪身从持握点向身后低位延伸
角色进入后侧蓄枪姿态
```

禁止出现：

```text
枪尖上翘
枪尖停在身体侧面
枪尖停在身前
枪身像剑气一样甩成大月牙
```

## 判定区域

```text
主要判定：身前正上到身体侧面的回扫路径
次要判定：身体侧面到身后低位的收枪路径
```

第三段可以扫到身后，但身后低位蓄枪姿态本身不作为持续高伤害区域。

## 视觉表现

```text
枪尖细弧轨迹
枪杆半透明残影
绕身路径清晰
身后低位出现收势闪光
```

## 命中反馈

```text
横向冲击线
扫开感
击退略强
伤害不作为主爆发
```

---

# 9. 动作四 A：地面进尺

## 触发条件

第四段开始瞬间，玩家在地面。

## 动作定义

保持第三段结束的身后低位蓄枪姿态，短暂蓄力后向身前长距离刺出。

这不是普通突刺，而是：

```text
借第三段回扫后的后侧蓄势，突然向前破阵。
```

## 蓄力姿态

蓄力开始时完全继承第三段结束姿态：

```text
枪尖在身后低位
枪尖微微下压
枪身不提前抬起
枪身不提前向前移动
角色重心略低
```

蓄力阶段允许出现：

```text
枪尖聚光
枪身流光
脚下尘线
轻微震动
```

蓄力阶段不允许出现：

```text
枪尖上抬
枪身变成斜上
提前出枪
重新摆成默认持枪姿态
```

## 释放路径

```text
身后低位蓄枪
→ 枪身经过角色侧线
→ 身前中线
→ 身前远端贯穿
```

## 判定区域

```text
主伤害区：角色身前长直线
最高反馈区：枪尖远端
身后抽枪段：仅表现动作残影，不作为主伤害区
```

## 视觉表现

```text
身后低位蓄光
由后向前拉直的速度线
长直线枪芒
枪尖强闪
敌人身后的贯穿线
连续命中小爆点
```

## 命中反馈

```text
贯穿
破阵
直线爆发
```

# 10. 动作四 B：空中抡枪下砸

## 触发条件

第四段开始瞬间，玩家在空中。

## 动作定义

空中第四段不是水平进尺，而是整根枪从身后向身前绕过人物头顶后下砸。

关键定义：

```text
整根枪作为刚体运动
以枪尾 / 持握端为旋转原点
从身后低位开始
绕过人物头顶
砸到人物身前
最终枪身微微下压
```

## 蓄力姿态

蓄力开始时继承第三段结束姿态：

```text
枪尖在身后低位
枪尖微微下压
角色在空中保持压势
枪身不提前举过头顶
```

## 抡枪路径

```text
身后低位蓄枪
→ 以枪尾 / 持握端为原点开始旋转
→ 整根枪从后往前绕过人物头顶
→ 枪尖落向人物身前
→ 枪身最终微微下压
```

路径观感：

```text
像以枪尾为圆心画一个大半圆
不是只有枪尖甩出弧线
不是普通下刺
不是剑类下劈
```

## 判定区域

```text
主要判定：人物身前下砸区域
辅助判定：抡枪经过头顶到身前的弧线区域
```

主伤害集中在下砸终点。

```text
抡枪路径：中等伤害或轻量命中
身前落点：主伤害
```

如果实现只做一次伤害，则伤害集中在身前下砸落点。

## 视觉表现

```text
身后低位蓄光
整根枪绕头顶的大半圆残影
枪尾 / 持握端有旋转原点感
枪尖砸到身前时出现爆光
短竖向冲击线
尘土
小地裂
```

## 结束姿态

下砸结束时：

```text
枪身位于人物身前
枪身微微下压
枪尖落在身前下方
动作有重击收束感
```

# 11. 特效统一规则

长矛特效分为四层：

```text
枪尖轨迹
枪身残影
命中爆点
环境反馈
```

优先级：

```text
枪尖最亮
枪身次亮
残影较淡
环境反馈最弱
```

不同段的特效强度：

```text
扎：弱
上挑：中弱
绕身回扫：中
地面进尺：强
空中抡枪下砸：强
```

禁止使用：

```text
巨大月牙刀光
宽厚剑气
360 度圆形斩
全身旋风
连续剑刃残影
```

允许使用：

```text
细长枪芒
枪尖光点
枪杆拖影
贯穿线
点状爆光
抡枪大半圆残影
落点尘土
小地裂
```

# 12. 武器特殊效果挂载规则

原有长矛的特殊效果保留，并挂载到新动作的合理位置。

通用规则：

```text
原本在枪尖触发的效果，挂载到当前动作枪尖位置。
原本命中触发的减益，仍在命中时触发。
原本发射的枪尖弹幕，优先挂载到扎、进尺或下砸落点。
原本的持续区域效果，挂载到对应动作路径或命中点。
```

示例定义：

```text
风暴长矛：
雷电效果跟随枪尖，进尺时表现最强。

暗黑长枪：
暗影焰减益保留，进尺命中时可强化暗影爆点表现。

蘑菇长矛：
蘑菇类路径效果可出现在回扫或进尺路径附近。

北极：
冰矛 / 雪花效果优先挂载到进尺和空中下砸终点。

恐怖关刀：
上挑和回扫可保留更夸张的长柄摆动感。
```

# 13. 最终动作定义

长矛类武器获得专属四段枪势连招。

```text
第一段：扎
快速向身前点刺，建立枪线。

第二段：上挑
枪尖从身前挑至身前正上，制造纵向起势。

第三段：绕身回扫
枪尖从身前正上绕身体外侧回到身后低位，枪尖微微下压，形成后侧蓄枪姿态。

第四段地面：进尺
保持第三段的身后低位蓄枪姿态，短暂蓄力后向身前长距离贯穿刺出。

第四段空中：抡枪下砸
保持第三段的身后低位蓄枪姿态，整根枪以枪尾 / 持握端为原点，从身后绕过人物头顶画出大半圆，砸到人物身前，结束时枪身微微下压。
```

这套动作的核心关系是：

```text
第三段负责把枪收到身后低位。
第四段负责借这个蓄势爆发。
```

地面爆发方向是：

```text
向前贯穿
```

空中爆发方向是：

```text
从后越头抡到身前下砸
```

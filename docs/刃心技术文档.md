# 刃心技术实现文档

本文档描述当前仓库中的实际实现，不再记录旧版方案或伪代码。

## 1. 实现范围

当前已经落地 12 个饰品：

```text
刃势线：
疾风剑符
破甲剑符
锐利剑符
三刃势徽章

重斩线：
断岳坠饰
延锋坠饰
崩锋徽章

蓄力线：
凝势沙漏
长锋晶坠
裂势印记
聚势纹章

最终饰品：
刃心
```

三条线的最终模型为：

```text
刃势线：支持的剑气命中后积累刃势，刃势临时提高近战速度、暴击率和近战伤害
重斩线：强化普通攻击四段连击中的第四段
蓄力线：强化右键蓄力斩的蓄满速度、距离、宽度和伤害
刃心：整合三刃势徽章、崩锋徽章和聚势纹章的能力，不额外添加隐藏机制
```

## 3. 饰品基类与冲突规则

所有刃心体系饰品都继承 `BladeAccessoryItem`。

基础属性：

```text
Item.width = 32
Item.height = 32
Item.accessory = true
Item.rare = ItemRarityID.Orange
Item.value = Item.sellPrice(gold: 2)
```

部分高级饰品覆盖稀有度和售价：

| 饰品 | 稀有度 | 售价 |
| --- | --- | --- |
| 三刃势徽章 | `LightRed` | 6 金 |
| 崩锋徽章 | `LightRed` | 6 金 |
| 聚势纹章 | `Pink` | 8 金 |
| 刃心 | `Yellow` | 20 金 |

当前冲突规则由 `BladeAccessoryItem.CanAccessoryBeEquippedWith()` 处理：

```text
刃心不能和任何刃心体系饰品同时装备。
每条线的终饰品不能和同线组件同时装备。
同一条线的组件之间可以同时装备。
不同条线的终饰品可以同时装备。
```

也就是说：

```text
疾风剑符 + 破甲剑符 + 锐利剑符：允许
三刃势徽章 + 任一刃势线组件：禁止
断岳坠饰 + 延锋坠饰：允许
崩锋徽章 + 任一重斩线组件：禁止
凝势沙漏 + 长锋晶坠 + 裂势印记：允许
聚势纹章 + 任一蓄力线组件：禁止
三刃势徽章 + 崩锋徽章 + 聚势纹章：允许
刃心 + 任一刃心体系饰品：禁止
```

## 4. 刃势线实现

刃势线不是常驻属性，而是命中叠层。

刃势状态保存在 `WeaponEffectsPlayer`：

```text
MaxBladeMomentumStacks = 8
BladeMomentumDecayInterval = 30
BladeMomentumNpcCooldown = 12
BladeMomentumStacks
```

饰品每 tick 通过 `RegisterBladeMomentum(...)` 注册本帧有效的刃势能力。`UpdateEquips()` 调用 `ApplyBladeMomentumStats()`，根据当前层数给玩家加近战属性。

### 4.1 刃势获取

刃势只从玩家拥有的 `SlashArcProjectile` 命中 NPC 时获取：

```text
SlashArcProjectile.OnHitNPC()
-> TryGrantBladeMomentum(player, target)
-> WeaponEffectsPlayer.TryGainBladeMomentum(Projectile, target)
```

限制规则：

```text
没有装备刃势相关饰品时不会获得刃势。
NPC 拥有的剑气不会给玩家刃势。
每个 SlashArcProjectile 最多授予 1 层刃势。
同一个 NPC 有 12 帧授予冷却。
装备刃势饰品后，支持的剑气命中 NPC 增加 1 层，最多 8 层。
```

原版武器兼容弹幕由 `VanillaMeleeProjectileEmitter` 发射，它们本身不会授予刃势；授予点只在本模组的 `SlashArcProjectile`。

### 4.2 刃势持续和衰减

`TryGainBladeMomentum()` 获得层数时会刷新持续时间：

```text
BladeMomentumStacks = Math.Min(8, BladeMomentumStacks + 1)
_bladeMomentumDuration = 当前饰品注册的持续时间
_bladeMomentumDecayTimer = 30
```

`UpdateBladeMomentum()` 每 tick 更新：

```text
没有刃势饰品处于激活状态：层数清零。
有层数且持续时间还没结束：持续时间递减。
持续时间结束后：每 30 帧减少 1 层。
```

当前没有独立的刃势 UI，也没有额外的满层触发技；刃势只作为临时属性来源。

### 4.3 刃势饰品数值

| 饰品 | 注册内容 | 满层效果 |
| --- | --- | --- |
| 疾风剑符 | 持续 120 帧，每层 +4% 近战速度，最多 8 层 | +32% 近战速度 |
| 破甲剑符 | 持续 180 帧，每层 +2% 近战暴击率，最多 4 层 | +8% 近战暴击率 |
| 锐利剑符 | 持续 180 帧，每层 +2% 近战伤害，最多 5 层 | +10% 近战伤害 |
| 三刃势徽章 | 持续 180 帧，同时注册以上三类属性 | +32% 近战速度、+8% 暴击率、+10% 近战伤害 |
| 刃心 | 调用 `ApplyMomentumStats(player)` | 等同三刃势徽章 |

普通剑气频率会读取玩家当前近战速度：

```text
SlashChannelProjectile.NormalSlashInterval
= _useAnimation * NormalSlashIntervalMultiplier / player.GetAttackSpeed(DamageClass.Melee)
```

因此疾风剑符、三刃势徽章、刃心和原版近战速度来源都会影响 WeaponEffects 的普通剑气频率。

## 5. 重斩线实现

重斩线通过 `WeaponEffectsPlayer.RegisterHeavySlash(float damageMultiplier, float lengthMultiplier)` 注册：

```text
FourthSlashDamageMultiplier = Math.Max(当前值, damageMultiplier)
FourthSlashLengthMultiplier = Math.Max(当前值, lengthMultiplier)
```

这意味着多个重斩组件同时装备时取每项最大值，不做乘法叠加。

### 5.1 触发点

`SlashChannelProjectile.FireComboSchemeASlash()` 创建普通剑气前消费当前连击段数：

```text
comboStepIndex = effectsPlayer.ConsumeNextSlashComboStep()
```

当前连段来自 `Compact3DComboSchemeA`，共 4 段，第四段索引为 `3`。只有 `comboStepIndex == 3` 时应用重斩加成。

### 5.2 第四段计算

第四段的实际长度公式：

```text
length =
    _weaponLength
    * LegacyAverageLengthScale
    * step.LengthScale
    * FourthSlashLengthMultiplier
    * weaponProfile.Shape.LengthScale
```

第四段的实际伤害公式：

```text
damage = round(NormalSlashDamage * FourthSlashDamageMultiplier)
```

当前只改第四段的伤害和距离，不新增专属视觉字段，也不需要给 `SlashArcProjectile` 增加额外同步字段。

### 5.3 重斩饰品数值

| 饰品 | 注册值 | 效果 |
| --- | --- | --- |
| 断岳坠饰 | `RegisterHeavySlash(1.30f, 1.00f)` | 第四段伤害 +30% |
| 延锋坠饰 | `RegisterHeavySlash(1.00f, 1.80f)` | 第四段距离 +80% |
| 崩锋徽章 | `RegisterHeavySlash(1.35f, 1.80f)` | 第四段伤害 +35%，距离 +80% |
| 刃心 | `RegisterHeavySlash(1.35f, 1.80f)` | 等同崩锋徽章 |

## 6. 蓄力线实现

蓄力线通过 `WeaponEffectsPlayer.RegisterChargeAccessory(...)` 注册：

```text
ChargeReadyFrameMultiplier = Math.Min(当前值, clamp(readyFrameMultiplier, 0.01f, 1f))
ChargeLengthBonusAtHalf = Math.Max(当前值, lengthBonusAtHalf)
ChargeLengthBonusAtFull = Math.Max(当前值, lengthBonusAtFull)
ChargeDamageBonusAtHalf = Math.Max(当前值, damageBonusAtHalf)
ChargeDamageBonusAtFull = Math.Max(当前值, damageBonusAtFull)
ChargeWidthBonusAtFull = Math.Max(当前值, widthBonusAtFull)
```

蓄满速度取最小倍率，距离、伤害和宽度取最大值。多个蓄力组件同时装备时会自然合并为聚势纹章的效果。

### 6.1 蓄满时间

`ChargedSlashProjectile` 中的基础时间：

```text
MinChargeFrame = useAnimation * ChargeMinDurationMultiplier
ChargeReadyFrame = max(MinChargeFrame, useAnimation * ChargeMaxDurationMultiplier)
```

最终蓄满时间：

```text
EffectiveChargeReadyFrame =
    max(MinChargeFrame, round(ChargeReadyFrame * ChargeReadyFrameMultiplier))
```

当前凝势沙漏、聚势纹章和刃心使用 `0.70f` 倍率，也就是满蓄时间缩短 30%，不是固定减少若干 tick。

`ChargeProgress`、满蓄视觉和蓄力条都使用 `EffectiveChargeReadyFrame`。

### 6.2 距离、伤害和宽度

释放时如果蓄力时间未达到 `MinChargeFrame`，不会创建蓄力剑气。

达到最短蓄力后释放时：

```text
progress = ChargeProgress
lengthBonus = EvaluateHalfFullBonus(progress, ChargeLengthBonusAtHalf, ChargeLengthBonusAtFull)
damageBonus = EvaluateHalfFullBonus(progress, ChargeDamageBonusAtHalf, ChargeDamageBonusAtFull)
widthBonus = ChargeWidthBonusAtFull * progress

lengthScale = CurrentLengthScale * (1 + lengthBonus)
damage = round(Projectile.damage * CurrentDamageScale * (1 + damageBonus))
thickness = 0.45f * (1 + widthBonus)
```

`EvaluateHalfFullBonus()` 在 0 到半蓄、半蓄到满蓄两段线性插值。

伤害加成是在当前蓄力最终伤害基础上乘法增加：

```text
Projectile.damage * CurrentDamageScale * (1 + damageBonus)
```

### 6.3 蓄力饰品数值

| 饰品 | 注册值 | 效果 |
| --- | --- | --- |
| 凝势沙漏 | `RegisterChargeAccessory(0.70f, 0f, 0f, 0f, 0f, 0f)` | 满蓄时间缩短 30% |
| 长锋晶坠 | `RegisterChargeAccessory(1f, 0.40f, 0.80f, 0f, 0f, 0.10f)` | 半蓄距离 +40%，满蓄距离 +80%，满蓄宽度 +10% |
| 裂势印记 | `RegisterChargeAccessory(1f, 0f, 0f, 0.10f, 0.25f, 0f)` | 半蓄伤害 +10%，满蓄伤害 +25% |
| 聚势纹章 | `RegisterChargeAccessory(0.70f, 0.40f, 0.80f, 0.10f, 0.25f, 0.10f)` | 综合三件蓄力组件 |
| 刃心 | 同聚势纹章 | 等同聚势纹章 |

## 7. 刃心实现

`BladeHeart.UpdateAccessory()` 当前只整合三条线最终能力：

```csharp
ApplyMomentumStats(player);

WeaponEffectsPlayer effectsPlayer = player.GetModPlayer<WeaponEffectsPlayer>();
effectsPlayer.RegisterHeavySlash(1.35f, 1.80f);
effectsPlayer.RegisterChargeAccessory(0.70f, 0.40f, 0.80f, 0.10f, 0.25f, 0.10f);
```

刃心没有额外的专属爆发、满刃势消耗、独立 UI 或隐藏触发条件。

## 10. 验证清单

```text
无饰品：
普通斩、第四段连击和右键蓄力保持原有表现。

疾风剑符：
支持的剑气命中后叠刃势；叠层后普通剑气频率随近战速度提高。

破甲剑符：
支持的剑气命中后叠刃势；最多提供 +8% 近战暴击率。

锐利剑符：
支持的剑气命中后叠刃势；最多提供 +10% 近战伤害。

三刃势徽章：
同一套刃势层数同时提高近战速度、暴击率和伤害。

断岳坠饰：
只有第四段连击斩击伤害提高 30%。

延锋坠饰：

崩锋徽章：
第四段连击斩击伤害提高 35%，距离提高 80%。

聚势纹章：

刃心：
等同三刃势徽章、崩锋徽章、聚势纹章的合并效果，并与所有刃心体系饰品冲突。
```

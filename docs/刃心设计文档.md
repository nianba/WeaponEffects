# 三势饰品体系

本文档描述当前已经落地的刃心饰品体系。

三条线分别是：

```text
刃势线：支持的剑气命中后积累刃势，刃势越高，普通作战能力越强
重斩线：强化普通攻击第四段 combo 斩击
蓄力线：强化右键蓄力斩
```

最终合成为：

```text
三刃势徽章
+ 崩锋徽章
+ 聚势纹章
= 刃心
```

核心体验是：

> **刃势负责“连续命中的成长反馈”，重斩和蓄力负责“把节奏打成爆发”，刃心把三条线收束成完整近战循环。**

---

# 一、总设计原则

这套饰品不是单纯常驻堆数值。当前实现保留了 WeaponEffects 的特色来源：

```text
支持的剑气命中
-> 获得刃势层数
-> 近战速度 / 暴击 / 伤害随层数提高
-> 普通攻击打到第四段 combo，或使用右键蓄力斩
-> 打出强化终结斩
```

当前版本不做满刃势消耗、独立 UI 或额外隐藏技能。这样实现范围更可控，也不会把普通攻击、第四段重斩和蓄力斩耦合成难以维护的复杂状态机。

---

# 二、刃势线：连续命中成长

## 刃势基础规则

刃势线专注于 **连续命中**。

当前规则：

```text
装备刃势线饰品后，支持的 SlashArcProjectile 命中 NPC 时获得 1 层刃势。
刃势最多 8 层。
每个 SlashArcProjectile 最多获得 1 层刃势。
同一个 NPC 有 12 帧授予冷却。
命中会刷新当前饰品提供的刃势持续时间。
持续时间结束后，每 30 帧衰减 1 层。
没有刃势线饰品激活时，刃势清零。
```

刃势奖励的是“稳定连续攻击”，不是靠多段碰撞瞬间刷满。当前没有独立刃势 UI，反馈主要来自攻速、暴击、伤害的手感变化。

---

```text

```

# 三、重斩线：强化第四段 combo

当前普通攻击使用四段循环连击，所以“第四段 combo 斩击”是重斩线的强化对象。

重斩线的核心是：

```text
第一斩、第二斩、第三斩用于铺垫。
第四斩变成真正的终结斩。
```

重斩只影响第四段剑气的伤害和距离，不额外发射飞剑或爆炸。这样手感直接，也不会挤占蓄力线的爆发定位。

---

## 1. 断岳坠饰

定位：**提高第四段 combo 伤害**

效果：

```text
第四段 combo 斩击造成更高伤害。
前三段攻击不受影响。
```

当前数值：

| 项目 | 数值 |
| --- | --- |
| 第四斩伤害 | +30% |
| 前三斩伤害 | 不变 |
| 触发条件 | 必须正常打到第四段 |

体验定位：

> 适合喜欢完整打完连段的玩家。

---

## 2. 延锋坠饰

定位：**提高第四段 combo 距离**

效果：

```text
第四段 combo 斩击的剑气距离和判定范围提高。
前三段攻击不受影响。
```

当前数值：

| 项目 | 数值 |
| --- | --- |
| 第四斩距离 | +80% |
| 前三斩距离 | 不变 |

体验定位：

> 让第四斩成为清怪、收尾、打远处敌人的关键一刀。

---

## 重斩线合成成品：崩锋徽章

合成：

```text
断岳坠饰
+ 延锋坠饰
= 崩锋徽章
```

效果：

```text
第四段 combo 斩击同时获得更高伤害和更远距离。
```

当前数值：

| 项目 | 数值 |
| --- | --- |
| 第四斩伤害 | +35% |

这里的设计重点是 **稳定、干净、有手感**。第四斩只需要变得更重、更远、更有压迫感。

# 四、蓄力线：强化右键蓄力

蓄力线专注于右键蓄力斩。

当前实现没有把蓄力设计成“更久才蓄满”。凝势沙漏和上级饰品的方向是 **缩短满蓄时间**，让玩家更稳定地打出蓄力斩。

---

## 1. 凝势沙漏

定位：**缩短蓄力时间**

当前数值：

| 项目 | 数值 |
| --- | --- |
| 满蓄时间 | 缩短 30% |

实现上是把满蓄帧数乘以 `0.70f`，并保留最短蓄力时间下限。

---

## 2. 长锋晶坠

定位：**提高蓄力斩距离**

效果：

```text
右键蓄力斩的剑气距离提高。
蓄力越满，距离提升越明显。
满蓄时剑气宽度略增。
```

当前数值：

| 项目 | 数值 |
| --- | --- |
| 半蓄距离 | +40% |
| 满蓄距离 | +80% |
| 满蓄剑气宽度 | +10% |

体验定位：

> 适合喜欢安全距离、打 Boss、处理飞行敌人的玩家。

---

## 3. 裂势印记

定位：**提高蓄力斩伤害**

效果：

```text
右键蓄力斩伤害提高。
蓄力越满，伤害提升越明显。
```

当前数值：

| 项目 | 数值 |
| --- | --- |
| 半蓄伤害 | +10% |
| 满蓄伤害 | +25% |
| 普通攻击伤害 | 不变 |

蓄力伤害提升在当前蓄力最终伤害基础上做乘法：

```text
最终伤害 = 基础伤害 * 当前蓄力伤害倍率 * (1 + 裂势印记加成)
```

# 五、最终饰品：刃心

三个子类最终饰品合成：

```text
三刃势徽章
+ 崩锋徽章
+ 聚势纹章
= 刃心
```

当前刃心只保留三者的能力，不额外添加隐藏技能：

```text
刃心 =
三刃势徽章能力
+ 崩锋徽章能力
+ 聚势纹章能力
```

刃心的目标阶段更接近机械 Boss 后到世纪之花后。当前配方没有加入断裂英雄剑、叶绿锭或三魂等阶段材料，只需要三条线的最终饰品在工匠作坊合成。

---

# 六、装备冲突

当前装备冲突规则：

```text
刃心不能和任何刃心体系饰品同时装备。
每条线的终饰品不能和同线组件同时装备。
同一条线的组件之间可以同时装备。
不同条线的终饰品可以同时装备。
```

这个规则让玩家在合成前可以组合组件体验三条线，也避免合成终饰品后继续叠同线能力导致强度膨胀。

# 七、当前体验总结

```text
疾风剑符：命中叠层后越打越快。
破甲剑符：命中叠层后暴击率提高。
锐利剑符：命中叠层后近战伤害提高。
三刃势徽章：同一套刃势同时提高攻速、暴击和伤害。

断岳坠饰：第四段连击更痛。
延锋坠饰：第四段连击更远。
崩锋徽章：第四段连击同时更痛更远。

凝势沙漏：蓄力斩更快蓄满。
长锋晶坠：蓄力斩距离和宽度提高。
裂势印记：蓄力斩伤害提高。
聚势纹章：蓄力线完整强化。

刃心：三刃势徽章、崩锋徽章、聚势纹章的合并版本。
```

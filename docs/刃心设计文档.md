# 三势饰品体系

三条线分别是：

```text
刃势线：连续命中越多，普通作战能力越强
重斩线：强化第四段 combo 斩击
蓄力线：强化右键蓄力斩
```

最终合成为：

```text
刃势终饰品
+ 重斩终饰品
+ 蓄力终饰品
= 刃心
```

核心体验是：

> **刃势负责“积累状态”，重斩和蓄力负责“释放爆发”，最终饰品把三者合成一个完整近战循环。**

---

# 一、总设计原则

这套饰品不要做成简单的：

```text
+10% 攻速
+10% 暴击
+10% 伤害
+20% 蓄力伤害
```

那样会变成普通数值饰品，缺少 WeaponEffects 的特色。

更好的设计是：

```text
连续命中 → 积累刃势
第四段 combo → 打出重斩
右键蓄力 → 打出蓄力斩
满刃势时使用重斩或蓄力斩 → 触发最终强化
```

也就是说，最终成品不是单纯把所有数值堆在一起，而是形成一个循环：

```text
普通攻击命中
→ 获得刃势层数
→ 攻速 / 暴击 / 伤害逐渐提高
→ 打到第四段 combo 或使用右键蓄力
→ 消耗或利用刃势打出强力终结
```

这样玩家会感觉自己在“打节奏”，而不是只是在穿一件加属性的饰品。

---

# 二、刃势线：连续命中成长

## 刃势基础规则

刃势线专注于  **连续命中** 。

建议定义：

```text
使用支持的近战剑气命中敌人时，获得 1 层刃势。
刃势最多 8 层。
一段时间未命中，刃势逐渐消失。
```

为了防止打蠕虫 Boss 或多段判定时层数瞬间叠满，建议加一条限制：

```text
同一次剑气最多获得 1 层刃势。
同一敌人在极短时间内不能重复提供刃势。
```

这样刃势奖励的是“稳定连续攻击”，而不是利用多段碰撞刷层数。

---

## 1. 疾风剑符

定位：**连续命中提高攻速**

效果：

```text
连续命中敌人时获得刃势。
每层刃势提高少量近战攻击速度。
```

建议数值：

| 项目         | 建议 |
| ------------ | ---- |
| 最大刃势     | 8 层 |
| 每层攻速     | +2%  |
| 满层攻速     | +16% |
| 刃势持续时间 | 2 秒 |
|              |      |

体验定位：

> 越打越快，适合快剑、短剑、妖刀、村正类武器。

视觉表现：

```text
每层刃势让玩家手部或武器边缘出现更密的细小刀痕粒子。
满层时，普通剑气尾部更短、更快、更干净。
```

---

## 2. 破甲剑符

定位：**连续命中提高暴击**

效果：

```text
连续命中敌人时获得刃势。
每层刃势提高近战暴击率。
```

建议数值：

| 项目         | 建议 |
| ------------ | ---- |
| 最大刃势     | 4 层 |
| 每层暴击率   | +2%  |
| 满层暴击率   | +8%  |
| 刃势持续时间 | 3 秒 |
|              |      |

体验定位：

> 更像“找破绽”的饰品，适合精准连击玩家。

视觉表现：

```text
刃势层数越高，剑气边缘越亮。
触发暴击时，命中点出现更尖锐的星形裂光。
```

---

## 3. 锐利剑符

定位：**连续命中提高伤害**

效果：

```text
连续命中敌人时获得刃势。
每层刃势提高少量近战伤害。
```

建议数值：

| 项目         | 建议 |
| ------------ | ---- |
| 最大刃势     | 5 层 |
| 每层伤害     | +2%  |
| 满层伤害     | +10% |
| 刃势持续时间 | 3 秒 |
|              |      |

体验定位：

> 越打越痛，适合大剑、重剑、宽剑气武器。

视觉表现：

```text
刃势层数越高，剑气颜色越厚重。
满层时，剑气中心出现短暂亮芯。
```

---

## 刃势线合成成品：三刃势徽章

合成：

```text
疾刃护符
+ 锐刃护符
+ 破刃护符
= 三刃势徽章
```

效果：

综合三个饰品

# 三、重斩线：强化第四段 combo

项目现在已经有四段普通攻击连段，所以“第四次 combo 斩击”是非常好的强化对象。它天然有节奏，也不需要新加复杂机制。

重斩线的核心是：

```text
第一斩、第二斩、第三斩用于铺垫。
第四斩变成真正的终结斩。
```

---

## 1. 断岳坠饰

定位：**提高第四段 combo 伤害**

效果：

```text
第四段 combo 斩击造成更高伤害。
前三段攻击不受影响。
```

建议数值：

| 项目       | 建议                           |
| ---------- | ------------------------------ |
| 第四斩伤害 | +30%                           |
| 前三斩伤害 | 不变                           |
| 触发条件   | 必须正常打到第四段             |
| 挥空处理   | 第四斩挥空则不返还加成         |
| 视觉       | 第四斩剑气更厚，命中有重击震动 |

体验定位：

> 适合喜欢完整打完连段的玩家。

注意，这个饰品不要直接提高所有攻击伤害，否则就会和刃势线的破刃护符重复。

---

## 2. 延锋坠饰

定位：**提高第四段 combo 距离**

效果：

```text
第四段 combo 斩击的剑气距离和判定范围提高。
前三段攻击不受影响。
```

建议数值：

| 项目       | 建议 |
| ---------- | ---- |
| 第四斩距离 | +55% |
| 前三斩距离 | 不变 |

体验定位：

> 让第四斩成为清怪、收尾、打远处敌人的关键一刀。

这个饰品的好处是手感非常明显。玩家会感觉：

> “前三刀是在接近和铺垫，第四刀是真的劈出去。”

---

## 重斩线合成成品：崩锋徽章

合成：

```text
断岳坠饰
+ 延锋坠饰
= 崩锋徽章
```

效果：

```text
第四段 combo 斩击同时获得更高伤害和更远距离。
第四斩命中敌人时产生明显重击反馈。
```

建议数值：

| 项目       | 建议 |
| ---------- | ---- |
| 第四斩伤害 | +35% |
| 第四斩距离 | +55% |

这里的设计重点是  **稳定、干净、有手感** 。

不要让第四斩额外发出一堆飞剑或爆炸。它只需要变得更重、更远、更有压迫感。

# 四、蓄力线：强化右键蓄力

蓄力线专注于右键蓄力斩。

这里有一个重要点：你说的“提高右键蓄力的时间”，我建议不要设计成“蓄满需要更久”，否则玩家会觉得这是负面效果。

更好的解释是：

```text
提高蓄力时间 = 提高最大蓄力窗口 / 延长满蓄保持时间 / 允许更稳定地蓄力。
```

也就是说，它让蓄力更可控，而不是单纯更慢。

---

## 1. 凝势沙漏

定位：**缩短蓄力时间**

建议数值：

| 项目     | 建议    |
| -------- | ------- |
| 满蓄时间 | -1.5 秒 |

## 2. 长锋晶坠

定位：**提高蓄力斩距离**

效果：

```text
右键蓄力斩的剑气距离提高。
蓄力越满，距离提升越明显。
```

建议数值：

| 项目     | 建议        |
| -------- | ----------- |
| 半蓄距离 | +10%        |
| 满蓄距离 | +25%        |
| 剑气宽度 | +5% 到 +10% |

体验定位：

> 适合喜欢安全距离、打 Boss、处理飞行敌人的玩家。

这个饰品最好只提高距离，不要同时大幅加宽，否则会和重斩线的第四斩范围感重复。

---

## 3. 裂势印记

定位：**提高蓄力斩伤害**

效果：

```text
右键蓄力斩伤害提高。
蓄力越满，伤害提升越明显。
```

建议数值：

| 项目         | 建议                   |
| ------------ | ---------------------- |
| 半蓄伤害     | +10%                   |
| 满蓄伤害     | +25%                   |
| 普通攻击伤害 | 不变                   |
| 视觉         | 满蓄命中时出现集中裂光 |

体验定位：

> 真正的蓄力爆发饰品。

注意，蓄力伤害提升在当前蓄力伤害的基础上做乘法，即当前蓄力最终伤害*1.25

---

## 蓄力线合成成品：聚势纹章

合成：

```text
凝势沙漏
+ 长锋晶坠
+ 裂势印记
= 聚势纹章
```

效果：

综合三者

# 五、最终最终饰品：刃心

三个子类最终饰品合成：

```text
三刃势徽章
+ 崩锋徽章
+ 聚势纹章
= 刃心
```

如果想让它更像后期大成饰品，可以加入后期材料：

```text
三刃势徽章
+ 崩锋徽章
+ 聚势纹章
+ 断裂英雄剑
+ 叶绿锭
= 刃心
```

也可以根据你希望的游戏阶段调整：

| 目标阶段       | 推荐附加材料             |
| -------------- | ------------------------ |
| 肉山前最终饰品 | 狱石锭、黑曜石、坠落之星 |
| 机械 Boss 后   | 神圣锭、三魂             |
| 世纪之花后     | 断裂英雄剑、叶绿锭       |
| 终局前         | 月亮碎片类材料           |

我个人建议把 **刃心** 放在 **机械 Boss 后到世纪之花后** 这个阶段。太早会压过原版战斗饰品，太晚又会让玩家体验不到完整体系。
